# CubeSat System

Grant-funded university capstone project involving the design and integration of multi-layer PCBs for satellite subsystems.

Public repository contains visuals only. Full design files are restricted.

---

## Motor Driver Board

**Key Highlights**
- 6-layer PCB stackup
- High-current BLDC phase routing
- Integrated current sensing
- Duel STM32 microcontroller integration

## Control / Sensor Board

**Key Highlights**
- IMU, gyro, and magnetometer integration
- I2C/SPI bus routing
- Compact layout under CubeSat constraints
- Embedded regulation and filtering
